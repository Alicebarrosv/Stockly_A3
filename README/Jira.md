# Gestão do projeto

Para a gestão do nosso projeto utilizamos a ferramenta **Jira**, que oferece integração direta com o **GitHub** e seus commits.  
Dessa forma, ao realizar um commit relacionado a uma tarefa, o Jira faz o **auto-preenchimento** das informações, facilitando o acompanhamento e rastreabilidade das atividades.
